

# ①課題名
どっちが大事なの？Bot(with gemini ver)

# ②課題内容（どんな作品か）
名詞を入力すると、画面上の彼女から「私と『名詞』どっちが大事なの？🥺」と返事が返ってくる
加えて、「疲れた」と打つと、彼女が改心し、心を包み込むような優しさで応答してくれる

# ③アプリのデプロイURL
- https://renren-geek.github.io/love-chat55/

# ④工夫した点・こだわった点
- 彼女のキャラをただ文体で変えるのではなく、モード切替（→geminiへ移行）で表現した点
- Gemini APIキーをフロントに置かずに、Node（server.js）経由で呼び出す構成にした点（キー漏洩リスクを回避）
- 一瞬でGeminiの無料枠を超えてしまうので、有料課金した点（1,000円を超えるとアラートが出るようにした）

# ⑤難しかった点・次回トライしたいこと（又は機能）
- 「画面の操作」「データの保存」「AIへの問い合わせ」が同時に動く仕組みになっていたので、見た目では同じように見えるトラブルでも、原因が違って発狂した。（例：「メッセージを送ったのにチャット画面に表示されない」という場合でも、①実はボタンが押されておらず画面の処理が動いていない？②データは保存されているが画面がそれを受け取れていない？③AIには問い合わせているが結果を画面に出していない？といったように、どこで止まっているのかが毎回違った。
そのため、今どこまで動いているのか？を画面/データ/サーバの順に確認しながら原因を探した。
